# edge-server-control

ESP32で実行可能なWebサーバー

利用可能なマイコンは以下です．
- ESP32

## Build Environment
- PlatformIO

## Installation

先にPlatformIO Coreをインストールしてください．
インストール方法は[こちら](https://docs.platformio.org/en/latest/core/installation/index.html)

```
cd <workspace>
git clone git@github.com:sgrsn/edge-server-control.git
cd edge-server-control
pio run -t upload
```
